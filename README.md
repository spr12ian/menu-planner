# Menu Planner

## Installation

``` sh
# get the code
git clone https://github.com/spr12ian/menu-planner.git

# work from the menu-planner directory
cd menu-planner

# install packages (clasp)
npm ci

# authorize clasp
npx clasp login

# If this is a new GAS project
npm run create
# Else
npm run clasp
```

Modify, as you wish, the whitelist of food websites in appscript.json:

"https://www.bbc.co.uk/food/",
"https://www.bbcgoodfood.com/",
"https://www.food.com/",
"https://www.nhs.uk/"
